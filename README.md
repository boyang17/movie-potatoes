# Movie Potatoes

A simple and modern movie tracking app that helps you keep track of what you have watched and what you want to watch.

## Try It Out

<a href="https://movie-potatoes.vercel.app/" target="_blank">Launch Movie Potatoes</a>

## Features

### Core Functionality

- Add movies to your **Watchlist** or mark movies as **Watched**
- Ability to move movies between the collections and remove them
- Persistent cloud storage with Supabase

### User Authentication
- Secure sign up and login
- User-specific movie data and session management

### User Experience

- Clean and responsive UI
- Fast and dynamic updates with React
- Real-time database syncing

## Technology

- **React JS**
- **Tailwind CSS**
- **Supabase**
- **PostgreSQL**
